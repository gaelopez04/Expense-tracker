# Apuntes Router

Para rutear componentes en paginas web (es decir cuando das click en un boton y te redirige por asi decirlo a otra URL pero de la misma pagina se utilida React-router-dom)

Para usarlo

npm install rect-router-dom

importar la librieria:

import { UseNavigate, Link, Routes, Route, BrowserRouter} from 'react-router-dom';
